fancytables.examples module: Examples
=====================================

This sub-module contains several minimal examples which demonstrate the usage
of the fancytables module. You are encouraged to look at their source code to
learn about basic and common use cases of fancytables. This page is merely
for short descriptions of each example method, so you know where to start if
you are interested in a certain feature. Most of these methods don't return
anything, but print their tables to the console.
